# Bigby: private, encrypted cloud storage from the UK

[Bigby](https://bigby.cloud) is a UK cloud storage service offering encrypted file storage and document collaboration. We are a privacy-first alternative to Google Drive, Dropbox and OneDrive, built for individuals, small businesses and teams who want their files kept private.

## In short

| | |
|---|---|
| **What** | Encrypted cloud storage and collaboration |
| **Where** | UK servers, UK data residency, GDPR compliant |
| **Who for** | Privacy-conscious individuals, small businesses and teams |
| **Privacy** | Files encrypted at rest. No scanning, no profiling, no AI training on your content |
| **Standards** | WebDAV support, open source stack |
| **Pricing** | Honest tiers from £3.99 per user per month, annual billing. No free tier, no data harvesting |

## Why Bigby exists

Big Tech storage is convenient, but the convenience is paid for with your data. Bigby takes the opposite approach. You pay a fair price for storage, we keep your files encrypted on UK infrastructure, and we have no interest in what is inside them.

We are independent and self-funded. No VC investors means no pressure to mine user data for growth. Just storage, done honestly.

## Plans

- **Storage**: private encrypted file storage. 100GB, 500GB or 1TB, from £3.99 per user per month
- **Office**: everything in Storage plus Collabora for documents, spreadsheets and presentations, from £5.99 per user per month
- **Group**: Office plans for teams of three or more at bulk pricing, from £4.99 per user per month
- **Storage Top-Up**: add 100GB to any plan for £1.49 per month

All prices are per user per month on annual billing. Available on web and desktop.
